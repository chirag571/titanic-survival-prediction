# Titanic Survival Prediction

## 📌 Project Overview
This project focuses on predicting passenger survival on the Titanic using data science techniques. The dataset was preprocessed, visualized using Seaborn and Matplotlib, and feature engineering was applied. A Logistic Regression model was used for classification, achieving approximately 80% accuracy. 

## 🧠 Model Used
- Logistic Regression

## 📊 Accuracy
- 80.44%

## 📂 Features
- Age
- Sex
- Pclass
- Fare
- FamilySize
